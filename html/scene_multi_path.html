Multipath Effect in Radar: Ground Reflection Analysis¶

Introduction¶

This notebook demonstrates multipath propagation—a fundamental phenomenon in radar where signals reach the target and return via multiple paths due to reflections from surfaces like the ground. Understanding multipath is critical for accurate target detection, as it can cause constructive or destructive interference, leading to amplitude fluctuations known as multipath fading.

What is Multipath?¶

Multipath occurs when radar signals propagate via multiple routes between transmitter, target, and receiver due to:

  • Ground reflections (most common)
  • Building/terrain reflections
  • Atmospheric ducting
  • Sea surface reflections

Each path has different:

  • Length → Phase shift
  • Amplitude → Path loss, reflection coefficient
  • Delay → Time of arrival

Multipath Propagation Paths¶

In this scenario (vertical multipath from ground):

  1. Direct Path: TX → Target → RX

    • Shortest path, no reflections
    • Primary signal component

  2. Two-Bounce Path: TX → Ground → Target → Ground → RX

    • Double ground reflection
    • Longest path, most attenuation

  3. TX Ground Reflection: TX → Ground → Target → RX

    • Transmit signal reflects off ground
    • Intermediate path length

  4. RX Ground Reflection: TX → Target → Ground → RX

    • Return signal reflects off ground
    • Intermediate path length

Interference and Multipath Fading¶

Phase Relationship:

Each path introduces phase shift based on path length difference ΔL:

$$\Delta\phi = \frac{2\pi \Delta L}{\lambda} = \frac{4\pi f_c \Delta L}{c}$$

Where:

  • $\Delta\phi$ = Phase difference (radians)
  • $\Delta L$ = Path length difference (m)
  • $\lambda$ = Wavelength (m)
  • $f_c$ = Carrier frequency (Hz)
  • $c$ = Speed of light (3×10⁸ m/s)

Interference Types:

  1. Constructive Interference: Δφ = 0°, 360°, 720°, ...

    • Signals add in-phase
    • Amplitude increase (up to +6 dB for equal paths)
    • Occurs at specific ranges

  2. Destructive Interference: Δφ = 180°, 540°, 900°, ...

    • Signals cancel (out-of-phase)
    • Amplitude decrease (nulls possible)
    • Occurs at intermediate ranges

Multipath Fading Pattern:

As target range changes, the phase difference varies, creating a lobing pattern with alternating peaks and nulls in received amplitude.

Two-Ray Ground Reflection Model¶

Simplified Model:

For radar at height $h_r$ and target at height $h_t$, separated by range $R$:

Path Difference:

$$\Delta L \approx \frac{2h_r h_t}{R}$$

(For $R >> h_r, h_t$)

Null Positions:

Destructive interference (nulls) occur when:

$$R_{null} = \frac{4h_r h_t}{\lambda(2n-1)}$$

Where $n = 1, 2, 3, ...$

First Null:

$$R_{null,1} = \frac{4h_r h_t}{\lambda}$$

Ground Reflection Coefficient¶

Fresnel Reflection:

Reflection from ground depends on:

  • Grazing angle: Angle between ray and ground
  • Polarization: Horizontal vs. vertical
  • Ground material: Permittivity ε_r, conductivity

Reflection Coefficient:

$$\Gamma = \frac{\sin\psi - \sqrt{\epsilon_r - \cos^2\psi}}{\sin\psi + \sqrt{\epsilon_r - \cos^2\psi}}$$

Where:

  • $\Gamma$ = Reflection coefficient
  • $\psi$ = Grazing angle
  • $\epsilon_r$ = Complex relative permittivity

For our scenario:

  • Ground: ε_r = 3.2 + 0.1j (typical asphalt/concrete)
  • Low grazing angles → High reflection (Γ ≈ -1 for perfect conductor)
  • Phase inversion: 180° phase shift on reflection

Impact on Radar Performance¶

Positive Effects:

  1. Increased Detection Range: Constructive interference can boost SNR
  2. Forward Scatter: Additional energy paths to target
  3. Diversity: Multiple paths for tracking

Negative Effects:

  1. Amplitude Fluctuation: Fading makes detection inconsistent
  2. False Alarms: Multipath returns may appear as separate targets
  3. Tracking Errors: Angle and range measurement degradation
  4. Null Zones: Blind ranges where detection fails

Mitigation Techniques¶

System Design:

  1. Frequency Diversity: Use multiple frequencies to avoid nulls
  2. Antenna Height: Optimize to shift nulls away from critical ranges
  3. Polarization: Exploit polarization differences
  4. Spatial Diversity: Multiple antennas at different heights

Signal Processing:

  1. Multipath Cancellation: Adaptive filtering
  2. Space-Time Processing: Exploit temporal/spatial structure
  3. Maximum Ratio Combining: Coherently combine paths
  4. CFAR: Adaptive threshold accounts for fading

Applications and Considerations¶

Automotive Radar:

  • Ground multipath significant at highway speeds
  • Affects pedestrian detection (low targets)
  • Requires robust processing algorithms

Aviation:

  • Runway surface reflections
  • Affects height measurement
  • Critical for landing systems

Coastal/Maritime:

  • Sea surface creates strong multipath
  • Wave motion adds Doppler spread
  • Challenging for surface vessel detection

Urban Environments:

  • Multiple reflection sources (buildings, vehicles)
  • Complex multipath scenarios
  • Requires 3D ray tracing

Case 1: Corner Reflector with Ground Plane¶

Simulate radar with both target and ground surface to demonstrate multipath effects.

Target and Ground Configuration¶

Corner Reflector:

  • Model: Trihedral corner reflector (high, stable RCS)
  • Initial Position: 300m range along x-axis
  • Velocity: -1 m/s (approaching radar)
  • Motion: Moves from 300m to 10m over 290 seconds

Ground Surface:

  • Model: 400m × 400m flat surface
  • Position: -0.5m (slightly below radar, simulating ground)
  • Complex Permittivity: ε_r = 3.2 + 0.1j
    • Real part (3.2): Typical for asphalt/concrete
    • Imaginary part (0.1j): Slight conductivity/loss
  • skip_diffusion: True (specular reflection only)

Multipath Propagation:

Four primary paths contribute:

  1. Direct: TX → Target → RX
  2. Two-bounce: TX → Ground → Target → Ground → RX
  3. TX reflection: TX → Ground → Target → RX
  4. RX reflection: TX → Target → Ground → RX

Ray Tracing and Processing¶

  • Density: 0.5 rays/wavelength² (adequate for flat surfaces)
  • Range FFT: Chebyshev window (60 dB sidelobe suppression)
  • Amplitude Extraction: Peak value per frame

# Import simulation and processing modules
from radarsimpy.simulator import sim_radar
from scipy import signal
import radarsimpy.processing as proc
import time

# Configure corner reflector target (moving)
target_1 = {
    "model": "../models/cr.stl",  # Trihedral corner reflector
    "location": (300, 0, 0),      # Initial position: 300m along x-axis
    "speed": (-1, 0, 0),          # Velocity: -1 m/s approaching
}

# Configure ground surface
target_2 = {
    "model": "../models/surface_400x400.stl",  # 400m × 400m flat surface
    "location": (0, 0, -0.5),                  # Position: 0.5m below radar
    "speed": (0, 0, 0),                        # Stationary
    "permittivity": 3.2 + 0.1j,                # Complex permittivity (asphalt/concrete)
    "skip_diffusion": True,                    # Specular reflection only
}

# Combine targets for multipath scenario
targets = [target_1, target_2]

# Start timing
tic = time.time()

# Simulate radar returns with multipath
# density=0.5: Ray tracing density for surfaces
# Simulates 290 frames (target moves 300m → 10m)
data = sim_radar(radar, targets, density=0.5)
baseband = data["baseband"]  # Complex I/Q samples

# Apply range FFT to extract range profiles
range_window = signal.windows.chebwin(radar.sample_prop["samples_per_pulse"], at=60)
range_profile = proc.range_fft(baseband, range_window)

# Extract peak amplitude per frame (max across range bins)
# Output: [290 frames, 1 channel] in dB
amp_multi = np.max(20 * np.log10(np.abs(range_profile)), axis=2)

# End timing
toc = time.time()
print("Exec time:", toc - tic, "s")

Results: Multipath Fading Pattern¶

Compare amplitude vs. range for both cases to visualize multipath interference effects.

Multipath Propagation Paths¶

Four primary paths contribute to interference:

  1. Path 1 (Direct): TX → Corner Reflector → RX

    • Shortest path
    • Strongest signal component
    • Reference phase

  2. Path 2 (Two-Bounce): TX → Ground → Corner Reflector → Ground → RX

    • Longest path
    • Most attenuation (two reflections)
    • Largest phase shift

  3. Path 3 (TX Reflection): TX → Ground → Corner Reflector → RX

    • Intermediate path length
    • One ground bounce (transmit side)

  4. Path 4 (RX Reflection): TX → Corner Reflector → Ground → RX

    • Intermediate path length
    • One ground bounce (receive side)

Interference Pattern¶

Constructive and Destructive Interference:

As target range changes, path length differences vary, causing phase relationships to cycle through:

  • Constructive: Paths add in-phase → amplitude peaks
  • Destructive: Paths cancel → amplitude nulls

Observable Pattern:

  • Free-space (blue): Smooth 1/R⁴ decay
  • With ground (orange): Oscillating "lobing" pattern superimposed on decay
  • Fluctuation amplitude: ±3 to 6 dB typical
  • Null depth: Can exceed 10-20 dB in deep fades
